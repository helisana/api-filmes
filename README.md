# React API Filmes


Este projeto usa a biblioteca React para consumir uma lista de filmes populares da API MovieDB.

Você também pode ver os detalhes de um filme ao clicar nele.

<h2>Tecnologias</h2>

<ul>
  <li>ReactJS</li>
  <li>React Router</li>
  <li>Styled Components</li>
  </ul>
